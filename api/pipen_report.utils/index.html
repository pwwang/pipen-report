pipen_report.utils.get_imagesize(filepath, cachedir)

</>

Return (width, height) for a given img file content

Code borrowed and adapted from imagesize package (https://github.com/shibukawa/imagesize_py/blob/master/imagesize/imagesize.py)

Note that for cloud files, for the image types that seeking is required to determine the dimensions, this function may not work properly.

Parameters
  • filepath (str | panpath.base.panpath) — a local or cloud file path
  • cachedir (str | panpath.base.panpath) — if filepath is a path from cachedir and it doesn't exist locally,we try to get it from the cloud
Returns (Tuple[int, int])

width and height of the image
